Project description - Avalon Smart Home (ASH) is the online platform for all your smart home devices, which includes five different types(i.e light, fan, heater, kitchen and audio). At ASH you can not only add your personalized devices to the central database but also can Update each device settings according to your preference.
Avalon Smart Home will be the all in one solution for maintaining the home devices, where you can perform operations like add, delete, check-status, update and much more. ASH works with the node.js javascript environment so that anyone can access it through the website.
Here we have one centralized MySql server to keep the device record, which consists of 12 columns (like device id, device name, device type and all the other device settings). With the help of Avalon Data server anyone can easily access all the information at one place.
